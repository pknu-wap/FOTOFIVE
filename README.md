FOTOFIVE
========
### 2020 web 5 project

#### 주제

* 사진 업로드/ 거래 웹사이트

* 상세 내용 : 개인이 찍은 사진을 업로드하거나, 사진으로 수익창출이 가능한 시스템의 웹사이트


#### 역할

* 프론트엔드 : 김동현 서지영
* 백엔드 : 안은성 정은정

## 사용 방법

1. git clone <URL> 또는 git Download
2. root 폴더에서 npm install
3. cd client 후 npm install
4. root 폴더에서 npm run backend
5. client 폴더에서 npm run start
